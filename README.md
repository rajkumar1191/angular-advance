Advanced Angular Core Showcase (playground)
==========================================

What's included:
- Standalone components demonstrating Signals, OnPush, dynamic component loading.
- Attribute & Structural directives.
- Pure & Impure pipes.
- ViewChild, ContentChild, Content Projection examples.
- Lifecycle hook logs and markForCheck usage.

How to run locally:
1. Install Angular CLI if you don't have it:
   npm install -g @angular/cli

2. Install dependencies:
   npm install

3. Serve the app:
   npm start
   (or: ng serve --open)

Notes:
- This is a minimal scaffold for demo purposes. Use Angular 16+.
- If you prefer StackBlitz, upload the project folder there or copy files into a new StackBlitz Angular project.
